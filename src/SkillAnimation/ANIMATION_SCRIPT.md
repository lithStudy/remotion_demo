# Agent Skill 原理动画脚本

## 动画总时长
- 总帧数：480帧 (16秒 @ 30fps)
- 无标题，直入主题

## 场景分解

### 场景1：文件系统扫描 (0-90帧, 3秒)
**目标**：展示 skill 文件的发现过程

**视觉元素**：
- 左侧：文件树结构，展示 `.cursor/skills/` 目录
- 右侧：空白区域（后续显示内容）
- 扫描动画：从顶部到底部的扫描线，高亮显示 skill 文件

**动画细节**：
- 0-30帧：文件树淡入
- 30-60帧：扫描线从上到下移动，发现 skill 文件
- 60-90帧：skill 文件高亮闪烁，表示被发现

**文本显示**：
- 文件路径：`.cursor/skills/remotion/SKILL.md`
- 状态提示："发现 Skill 文件"

---

### 场景2：Skill 文件读取 (90-180帧, 3秒)
**目标**：展示 skill 文件内容被读取

**视觉元素**：
- 左侧：文件树（保持显示）
- 右侧：文件内容窗口，逐行显示 skill 内容

**动画细节**：
- 90-120帧：文件内容窗口淡入
- 120-180帧：文件内容逐行显示（打字机效果）

**文本显示**：
```markdown
---
name: remotion-best-practices
description: Best practices for Remotion
---

# Remotion 最佳实践

所有动画必须使用 useCurrentFrame() 驱动...
```

---

### 场景3：Skill 解析 (180-270帧, 3秒)
**目标**：展示 skill 内容被解析成结构化数据

**视觉元素**：
- 左侧：原始 markdown 文件（缩小）
- 右侧：解析后的结构化数据（JSON 格式）

**动画细节**：
- 180-210帧：解析过程动画（数据流动效果）
- 210-270帧：结构化数据逐项显示

**文本显示**：
```json
{
  "name": "remotion-best-practices",
  "description": "Best practices for Remotion",
  "rules": [
    "所有动画必须使用 useCurrentFrame()",
    "禁止使用 CSS transitions",
    ...
  ]
}
```

---

### 场景4：提示词系统更新 (270-390帧, 4秒)
**目标**：展示系统提示词如何被更新

**视觉元素**：
- 上半部分：更新前的系统提示词（旧版本）
- 下半部分：更新后的系统提示词（新版本，包含 skill 内容）

**动画细节**：
- 270-300帧：显示更新前的提示词
- 300-330帧：skill 内容从解析区域流向提示词系统（数据流动动画）
- 330-390帧：提示词内容逐行更新，高亮显示新增的 skill 内容

**文本对比**：

**更新前**：
```
你是一个 AI 编程助手...
```

**更新后**：
```
你是一个 AI 编程助手...

[Skill: remotion-best-practices]
所有动画必须使用 useCurrentFrame() 驱动...
禁止使用 CSS transitions...
```

---

### 场景5：效果展示 (390-480帧, 3秒)
**目标**：展示 skill 如何影响 AI 行为

**视觉元素**：
- 左侧：用户请求示例
- 右侧：AI 响应（展示 skill 的影响）

**动画细节**：
- 390-420帧：用户请求淡入
- 420-450帧：AI 处理动画（思考过程）
- 450-480帧：AI 响应显示，高亮显示受 skill 影响的部分

**示例对话**：

**用户**：
```
帮我创建一个淡入动画
```

**AI 响应**（受 skill 影响）：
```
使用 useCurrentFrame() 和 interpolate() 实现淡入：

const opacity = interpolate(frame, [0, 30], [0, 1], {
  extrapolateLeft: 'clamp',
  extrapolateRight: 'clamp'
});
```

## 视觉设计规范

### 颜色方案
- 背景：深色主题 (#1a1a2e)
- 主色：青色 (#64ffda) - 用于高亮和强调
- 辅助色：蓝色 (#79c0ff) - 用于次要元素
- 文本：浅灰色 (#c9d1d9)
- 文件树：深灰色背景 (#0d1117)

### 动画风格
- 使用 spring 动画实现平滑过渡
- 使用 interpolate 实现线性动画
- 数据流动使用粒子或线条动画
- 高亮使用脉冲效果

### 字体
- 代码/文件内容：等宽字体 (monospace)
- 标题/说明：系统默认字体

---

## 技术实现要点

1. **文件树组件**：使用 SVG 或 div 绘制文件树结构
2. **打字机效果**：使用逐字符显示实现
3. **数据流动动画**：使用 SVG path 动画或粒子效果
4. **对比显示**：使用上下分屏展示 before/after
5. **高亮效果**：使用背景色变化和边框动画
