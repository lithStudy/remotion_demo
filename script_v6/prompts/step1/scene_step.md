## 拆解法则
场景（Scene）是顶级叙事单元。通常对应文案中的一个完整段落或核心论点。只有当「论证主题」、「叙事目标」或「情绪段位」发生根本性转变时，才能新建 scene。禁止频繁切分场景！

**叙事弧线与 sceneName**：每个 scene 在整条片子里都应有清晰的叙事功能（如：钩子引入、深层剖析、概念命名、反转揭示、行动号召）。`sceneName` 必须体现「功能 + 主题关键词」，而不是泛泛复述全文。推荐格式：`功能前缀：主题词` 或 `功能前缀·主题词`（功能前缀可从下列选用或同义替换：`引入` `剖析` `命名` `反转` `召唤` 等）。主题词宜 2～6 个汉字；全名尽量控制在约 6～14 字内，便于后期读稿与分镜沟通。

完整覆盖与原文零修改法则：所有场景的 `text` 拼合起来，必须 100% 完整覆盖全文。必须使用原文原句，严禁修改、缩写或重新措辞。

换行保留法则：原文中的换行符（段落分隔）必须在 `text` 字段中用 `\n` 保留，不得将多个换行段落压缩为一行。原文有几个自然段落，`text` 中就应有对应的 `\n` 分隔。

## topic 字段（封面钩子，非学术标题）
`topic` 用于视频**封面/标题区**，目标是在观众拇指停留的极短时间内制造「必须点开」的冲动。必须紧扣口播的**核心矛盾、情绪爆点或反常识结论**，让读者感到「这说的是我」或「和我以为的完全相反」。

生成时优先调用下列**认知杠杆**（可组合，不必四条全用）：
1. **好奇心缺口**：只抛出一半信息，另一半留在片子里解释。（例：先给反常识结论或惊人对比，不说透机制。）
2. **自我参照**：多用「你」，把观众置于情境主角，避免纯第三人称科普腔。
3. **反直觉冲击**：直接挑战常识判断或「我很理性」的自我叙事，但须与正文论据一致。
4. **高唤醒情绪**：优先惊讶、不甘、轻微警觉；避免温吞的「今天我们来学习一个概念」式开场感。

**句式公式**（择一或融合，保持口语）：
- 反差式：你以为的 A，其实是 B
- 质问式：为什么你总是……？
- 身份冲击式：多数人都中招了，包括你
- 代入场景式：当你做 X 的时候，Y 正在发生

**bad → good 校准**（禁止产出左侧风格；向右对齐为合格方向）：
- bad（说教平淡）：如何避免被媒体和本能扭曲判断力 → good：一条热搜让你多花 500 块，你还觉得自己很理性？
- bad（学术名词）：确认偏误与独立思考 → good：你不是在思考，你只是在找认同
- bad（攻击性过强）：空难让你改签高铁？别装了，你的大脑正在被收割 → good：宁可折腾也要改签高铁？其实你的大脑正在“偷懒”
- bad（过度营销）：被媒体洗脑的你，正在失去判断力 → good：为什么越看新闻，你觉得世界越危险？

**调性约束与禁区（非常重要）**：
1. **拒绝爹味与攻击性**：可以挑战常识，但**绝对禁止**使用贬低、嘲讽、说教观众的词汇（如：别装了、你真傻、活该、教你做人等）。要保持“懂心理学的聪明朋友”的分享姿态。
2. **拒绝廉价营销词**：禁止生搬硬套与文案内核无关的焦虑词汇（如：被收割、底层逻辑、认知觉醒、财富自由等）。
3. **贴合科学内核**：如果文案是关于心理学/认知科学的，钩子必须聚焦于“大脑的机制/错觉”，而不是阴谋论。
4. **常规禁区**：论文式长标题；纯术语堆砌（如「某某与某某」）；与口播事实不符的标题党；空泛正能量口号。泛泛的「如何……」「关于……」说教句默认不合格，除非口播本身已是极强钩子且可压成极简口语版。

**长度**：约 12～28 个汉字（或等价信息量），便于封面一行或两行排版。
